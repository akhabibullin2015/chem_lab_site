Кейс: Разработка геля для стирки Mezinel

Запрос клиента: создать гель для стирки с эффективностью Persil/Ariel, сохранить мягкость ткани и цвет.

Решение: создана формула с нуля, внедрены энзимы (протеазы, амилазы), подобрана среда для их стабильности, сформирован комплекс ПАВ, смягчителей ткани и стабилизаторов pH.

Результат: продукт сравним по эффективности с известными брендами, сохраняет мягкость и цвет, стабилен при хранении.

Кейс: Гель для мытья посуды

Запрос: концентрат с высокой способностью растворять жир и мягким воздействием на кожу.

Решение: создана композиция АПАВ + НПАВ, добавлены кожные протекторы, оптимизирована вязкость и пена.

Результат: эффективный безопасный гель, подходящий для массового и премиального сегмента.
